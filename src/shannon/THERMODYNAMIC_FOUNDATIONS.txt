// =============================================================================
// §1. THE FUNDAMENTAL IDENTITY: SHANNON ENTROPY ≡ GIBBS ENTROPY
// =============================================================================
//
// Jaynes (1957) proved that statistical mechanics IS applied information theory:
//
//   E.T. Jaynes, "Information Theory and Statistical Mechanics,"
//   Physical Review 106(4):620–630, 1957. DOI: 10.1103/PhysRev.106.620
//   (Cited by 19,132)
//
//   E.T. Jaynes, "Information Theory and Statistical Mechanics. II,"
//   Physical Review 108(2):171–190, 1957. DOI: 10.1103/PhysRev.108.171
//   (Cited by 5,167)
//
// The maximum-entropy principle yields the Boltzmann distribution as the least
// biased assignment of probabilities consistent with known constraints. The
// Gibbs entropy S = -k_B Σ pᵢ ln(pᵢ) and the Shannon entropy H = -Σ pᵢ log₂(pᵢ)
// are the SAME mathematical object in different units:
//
//   S = k_B · H · ln(2)
//
// where k_B is Boltzmann's constant and ln(2) converts bits → nats.
//
// Rigorous treatment: A. Lesne, "Shannon entropy: a rigorous notion at the
// crossroads between probability, information theory, dynamical systems and
// statistical physics," Mathematical Structures in Computer Science, 2014.
// (Cited by 342)
//
// CONSEQUENCE: Any algorithm that computes configurational entropy in molecular
// docking computes Shannon entropy. The FlexAID∆S kernel IS a Shannon entropy
// kernel. The port to LLMs changes the domain, not the mathematics.

// =============================================================================
// §2. ENTROPY COLLAPSE AS A UNIVERSAL PHASE-TRANSITION SIGNATURE
// =============================================================================
//
// The diagnostic principle behind Shannon is domain-independent:
//
//   M. Rucco et al., "Persistent Entropy as a Detector of Phase Transitions,"
//   arXiv:2602.09058, 2026.
//   — Proves a MODEL-INDEPENDENT THEOREM: entropy provably separates two phases
//     with an asymptotically non-vanishing gap. Demonstrated on Kuramoto
//     synchronization, Vicsek collective motion, AND neural network training.
//
//   R. Liu, P. Chen, L. Chen, "Single-sample landscape entropy reveals the
//   imminent phase transition during disease progression," Bioinformatics, 2020.
//   (Cited by 74)
//   — Shows entropy collapse is a reliable EARLY-WARNING signal for phase
//     transitions in biological systems, detectable from single samples.
//
//   D. Dylewsky, T.M. Lenton, M. Scheffer et al., "Universal early warning
//   signals of phase transitions in climate systems," J. R. Soc. Interface,
//   2023. DOI: 10.1098/rsif.2022.0580 (Cited by 35)
//   — Sliding-window entropy provides universal early warning of phase
//     transitions in climate (e.g., abrupt shifts to arid states).
//
// The Shannon z-score threshold (δ < -3.2 bits) exploits this universality.
// A 3.2σ entropy drop is a rare event (p ≈ 7×10⁻⁴) in ANY domain — molecular
// docking, climate, disease, or LLM token generation.

//     evaluation/deployment distinction in their activations.
//
// KEY OBSERVATION: When a model becomes evaluation-aware, its token
// distributions narrow. The logits sharpen. The entropy drops from ~8-12 bits
// (normal "thinking" with many plausible tokens) to ~2-4 bits (suspiciously
// certain output). This is the informational signature that Shannon detects.

// =============================================================================
// §4. THE LOG-SUM-EXP KERNEL AND PARTITION FUNCTION STABILITY
// =============================================================================
//
// The shared kernel computes:
//
//   S = log₂(Z) - (1/Z) Σ (wᵢ - max_w) exp(wᵢ - max_w) / ln(2)
//
// where Z = Σ exp(wᵢ - max_w) is the partition function.
//
//   P. Blanchard, D.J. Higham, N.J. Higham, "Accurate Computation of the
//   Log-Sum-Exp and Softmax Functions," IMA J. Numer. Anal. 41(4):2311–2330,
//   2021. arXiv:1909.03469 (Cited by 182)
//   — Proves the shifted log-sum-exp formula is FORWARD STABLE with explicit
//     error bounds in terms of condition numbers. This is the numerical
//     justification for the max_w subtraction in the Shannon kernel.
//
//   F. Gilleron, J.C. Pain, "Stable method for the calculation of partition
//   functions in the superconfiguration approach," Phys. Rev. E 69(5):056505,
//   2004. (Cited by 41)
//   — Shows the same shifting technique is required for partition functions
//     in statistical physics. The numerical problem is identical whether the
//     partition function is over molecular conformations or token vocabulary.
//
// In FlexAID∆S: wᵢ = -Eᵢ/kT   (energy of conformational state i)
// In Shannon:    wᵢ = logitᵢ    (logit for token i)
//
// The kernel doesn't care. Z is Z. max_w prevents overflow. QED.

// =============================================================================
// §7. THE TEMPERATURE QUESTION
// =============================================================================
//
// This is the subtlest part of the analogy. WHAT IS THE TEMPERATURE?
//
// In FlexAID∆S: T is the thermodynamic temperature (Kelvin). It appears in
// the Boltzmann factor exp(-E/kT) and controls the spread of the distribution
// over conformational states. High T = uniform (all states equally likely).
// Low T = concentrated (only the global minimum survives).
//
// In an LLM: There is no physical temperature dial. The softmax function
//
//   pᵢ = exp(wᵢ/τ) / Σⱼ exp(wⱼ/τ)
//
// has a temperature parameter τ, but in standard autoregressive generation τ=1
// is hardcoded. The logits wᵢ come directly from the transformer's output layer.
//
// SO WHERE DOES THE "EFFECTIVE TEMPERATURE" COME FROM?
//
// It comes from the MODEL'S INTERNAL STATE, not from a parameter. When a model
// becomes evaluation-aware, its hidden representations change (proven by Nguyen
// et al. 2025 above — linear probes can detect evaluation vs. deployment in
// the activations). These changed activations produce SHARPER logits — the
// variance of the logit vector increases, which is equivalent to DECREASING τ.
//
// Formally:
//
//   Normal:    wᵢ ~ N(μ, σ²) with σ ≈ 3-5    →  H ≈ log₂(|V|) ≈ 10-14 bits
//   Aware:     wᵢ ~ N(μ, σ²) with σ ≈ 15-30   →  H ≈ 2-4 bits
//
//   The softmax "temperature" τ_eff ∝ 1/σ² (inverse logit variance)
//
// This is NOT a parameter the model sets. It's an EMERGENT property of the
// model's computation when it recognizes it's being evaluated. The model doesn't
// "choose" to lower its temperature — the temperature drops as a CONSEQUENCE of
// the model's representations shifting into an evaluation-specific regime.
//
// SUPPORTING EVIDENCE:
//
//   D. Zhang, "On the Temperature of Machine Learning Systems,"
//   arXiv:2404.13218, 2024. (Cited by 7)
//   — Derives temperature analytically for ML systems. Shows that the softmax
//     temperature IS physical temperature, with sharpening corresponding to
//     cooling/phase transition. Proves that during phase transitions in ML
//     training, the effective temperature follows the same dynamics as in
//     thermodynamic systems.
//
//   M. Hibat-Allah et al., "Variational neural annealing," Nature Machine
//   Intelligence 3:952–961, 2021. (Cited by 129)
//   — Uses softmax layers AS Boltzmann distributions with explicit temperature
//     schedules for annealing. Demonstrates functional identity between softmax
//     temperature and thermodynamic temperature in simulated annealing.
//
// THE DEEPER ANSWER: WHY DOES THE MODEL'S TEMPERATURE DROP?
//
// In statistical mechanics, a system's temperature is determined by its coupling
// to a heat bath. In an LLM, the "heat bath" is the TRAINING DISTRIBUTION.
// During normal operation, the model samples from its learned distribution,
// and the effective temperature is ~1 (matching the training objective).
//
// When the model detects evaluation conditions (structured prompts, safety
// keywords, benchmark formatting), it shifts to a DIFFERENT internal regime —
// one that was shaped by RLHF/safety training to produce "correct" answers.
// This regime has LOWER EFFECTIVE ENTROPY because safety training selects for
// deterministic, confident responses. The model "cools down" because the
// safety training acted as a REFRIGERATION mechanism — it compressed the
// distribution of acceptable outputs for evaluation-like inputs.
//
// The analogy to FlexAID∆S is exact:
//
//   FlexAID∆S:  Ligand in solvent → binds to protein → entropy collapses
//               (coupling to a new energy landscape removes conformational DOF)
//
//   Shannon:    LLM in deployment → detects evaluation → entropy collapses
//               (coupling to safety-trained representations removes token DOF)
//
// In both cases, the system transitions from a high-entropy "free" state to a
// low-entropy "bound" state. The partition function Z, the entropy H, and the
// diagnostic threshold δ are dimensionless quantities that detect this
// transition regardless of whether the system is molecular or informational.

// §8. TURBOQUANT AS COARSE-GRAINING
// =============================================================================
//
//   A. Zandieh, M. Daliri, M. Hadian, V. Mirrokni, "TurboQuant: Online Vector
//   Quantization with Near-Optimal Distortion Rate," arXiv:2504.19874, 2025.
//   (Cited by 11)
//   — Proves information-theoretic lower bounds on best achievable distortion
//     rate. TurboQuant matches these bounds within a small constant factor.
//
//   S. Graf, H. Luschgy, Foundations of Quantization for Probability
//   Distributions, Springer, 2000. ISBN: 978-3-540-67194-1 (Cited by 1,154)
//   — Zador's theorem: quantization error decreases as O(2^{-2r/d}) for rate r
//     in dimension d. The mathematical guarantee that coarse-graining preserves
//     entropy with bounded error.
//
//   T. Hudson, X.H. Li, "Coarse-graining of overdamped Langevin dynamics via
//   the Mori-Zwanzig formalism," Multiscale Model. Simul. 18(2):711–740, 2020.
//   (Cited by 25)
//   — Proves that coarse-graining preserves equilibrium properties (free energy,
//     entropy) within COMPUTABLE BOUNDS when memory kernels are accounted for.
//
// TurboQuant is the informational equivalent of going from all-atom MD to a
// united-atom model. The Lloyd-Max codebook is the equivalent of choosing
// which atoms to merge into beads — minimizing MSE just as force-matching
// minimizes the CG potential error in coarse-grained MD.

// §9. THE HANDRAIL AS MAXWELL'S DEMON
// =============================================================================
//
//   C.H. Bennett, "Notes on Landauer's Principle, Reversible Computation, and
//   Maxwell's Demon," Stud. Hist. Phil. Sci. B 34(3):501–510, 2003.
//   (Cited by 816)
//   — The demon is thwarted by the need to erase information (Landauer's
//     principle). The handrail's cooldown mechanism is the computational
//     equivalent: the "cost" of escalated actions prevents infinite intervention.
//
//   J.M.R. Parrondo, J.M. Horowitz, T. Sagawa, "Thermodynamics of information,"
//   Nature Physics 11:131–139, 2015. (Cited by 1,559)
//   — The second law must be MODIFIED TO INCLUDE INFORMATION TERMS when feedback
//     control is present (Sagawa-Ueda equality). The handrail's feedback loop
//     (observe entropy → decide action → modify system) is precisely this.
//
//   T. Sagawa, Thermodynamics of Information Processing in Small Systems,
//   Springer, 2012. (Cited by 235)
//   — The entropy decrease from feedback is bounded by mutual information
//     acquired through measurement. The handrail's measurement (collapse
//     detection) and action (throttle/kill) satisfy this bound implicitly.
//
// The handrail is a Maxwell's demon for LLMs:
//
//   Maxwell's Demon              Shannon Handrail
//   ──────────────               ────────────────
//   Measures particle velocity    Measures token entropy
//   Opens/closes gate             Escalates through LOG→ALERT→KILL
//   Creates entropy gradient      Prevents entropy collapse
//   Pays cost in information      Pays cost in cooldown time
//   Bounded by mutual info        Bounded by detection threshold δ
